# Misconduct Case Management Tool (Client-Side)

A lightweight, browser-based tool to manage academic misconduct cases, generate documentation, and export case records.

## 🔧 Features

* 📄 Generate structured case records (DOCX)
* 📦 Download complete case package (ZIP)
* 🧾 Export and import case data (JSON)
* 🖨️ Print / Save as PDF
* ✉️ Generate emails (student communication templates)
* 📊 Panel summaries and case reports

## 🚀 Live Usage

Once deployed via GitHub Pages, the app runs entirely in the browser:

* No installation required
* No backend/server needed
* No data leaves the user’s machine

## 📁 Outputs

The tool allows you to generate:

* Case Record (DOCX)
* Case Package (ZIP)
* JSON backup file
* Email drafts (initial contact, outcome)
* Panel summaries (short + full)
* Register and summary text

## 🔐 Data Privacy

* All processing is done locally in the browser
* No data is uploaded to any server
* Suitable for handling sensitive academic data (use responsibly)

⚠️ Do NOT upload real student data to GitHub.

## 🛠️ How to Use

1. Open the app in your browser
2. Enter case details
3. Choose one of the export options:

   * Download DOCX
   * Download Case Package
   * Save JSON
   * Print / Save PDF

## 💻 Development

This is a fully client-side application built with:

* HTML
* CSS
* JavaScript

To update:

1. Edit `index.html`
2. Commit changes
3. Push to GitHub
4. GitHub Pages updates automatically

## 📦 Deployment (GitHub Pages)

1. Upload files to repository
2. Go to **Settings → Pages**
3. Select:

   * Branch: `main`
   * Folder: `/ (root)`
4. Access via:
   https://iftikharafridi.github.io/academic-misconduct-policy-app/
